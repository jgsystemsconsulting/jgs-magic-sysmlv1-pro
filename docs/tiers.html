Licence tiers

Tiers are about what your AI agent can do with the live model, not about anything that runs in the cloud. Every install starts at Free, which needs no licence file. This pro extension JAR is the artifact that unlocks the Pro tier, and licence checks happen entirely on your machine (there is no licence server to reach).

§01 · What each tier unlocks

FREE

Read & audit

Free is read and audit: your agent can query, navigate, audit, and report on the open model. No licence file is required, and no writes are possible at this tier.

PRO

Author & edit

Pro is author and edit: full write access to create and modify blocks, ports, requirements, relationships, and diagrams through your agent. This pro extension JAR (jgs-sysmlv1-pro.jar) is exactly what delivers Pro, and a valid licence file is required to activate it.

ENTERPRISE

Script & scale

Enterprise is script and scale: everything in Pro plus dangerous writes and Groovy scripting for power workflows. A valid licence file is required.
