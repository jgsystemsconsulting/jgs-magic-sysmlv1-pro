# Changelog

All notable changes to the SysML v1 MCP Bridge: Pro Extension are documented here.

## [0.1.0] - 2026-06-09

### Added
- Initial release of the pro-extension JAR (`jgs-sysmlv1-pro.jar`) for the
  SysML v1 MCP Bridge. The JAR is loaded dynamically at startup by the free
  base plugin and unlocks the licensed (write-tier and pro) tool set.
- `RELEASE-INFO.txt` recording the JAR SHA-256 digest, version, and intended tag.
